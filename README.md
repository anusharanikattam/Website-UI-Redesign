# 🎨 Website UI Redesign

> **UI/UX Internship Project**
> Designed and Developed by **K. Anusha Rani**

---

## 📌 Project Overview

This project focuses on redesigning an existing website to improve its **usability, accessibility, visual appeal, and overall user experience**. The redesigned interface follows modern UI/UX principles with a responsive layout that works smoothly on desktop, tablet, and mobile devices.

---

## 🎯 Objectives

* Improve website usability
* Create a clean and modern interface
* Enhance visual hierarchy
* Design responsive layouts
* Improve accessibility
* Provide a better user experience

---

## ✨ Key Features

* Responsive Navigation Bar
* Modern Hero Section
* Glassmorphism UI Design
* Heuristic Evaluation Section
* Mood Board
* Style Guide
* High-Fidelity Mockups
* Before & After UI Comparison
* Responsive Desktop & Mobile Layout
* Contact Section
* Smooth Animations
