Project Overview

A complete UI/UX redesign that solves real usability issues with a clean, modern aesthetic.

Objective

Redesign an existing website to create a cleaner, faster, and more user-friendly experience with a focus on visual hierarchy.

Tools

Outcome

Professional responsive interface with improved navigation, typography, spacing, and accessibility.

Heuristic Evaluation

Identified usability issues and the solutions applied in the redesign.

Problem 1: Confusing navigation and inconsistent menu placement.
Problem 2: Poor typography and weak visual hierarchy.
Problem 3: Low contrast colors affecting accessibility.
Solution: Improved layout, accessible color palette, responsive navigation, larger typography and modern components.
